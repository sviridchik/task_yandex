1) From github 

- pip install -r requirements.txt
- cd /sweets
if you want to run server
- python3 manage.py runserver 0.0.0.0:8080
if you want to run tests
- python3 manage.py test

2) Remote machine:
- sudo apt update
- sudo apt install python3-pip
- pip install -r requirements.txt
- cd sweets
- python3 manage.py runserver 0.0.0.0:8080
